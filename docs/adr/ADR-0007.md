# ADR-0007: Configuration & Secrets

## Status
Accepted

## Context
appsettings.Development.json suele terminar conteniendo secrets locales.

## Decision
No se commitean secretos.
- Local: User Secrets / variables de entorno.
- Docker: variables en compose (.env ignorado).
- Repo: appsettings.json con defaults y/o appsettings.Development.sample.json.

## Consequences
+ Seguridad.
+ Portabilidad a CI/CD.
- Requiere setup inicial para devs.
