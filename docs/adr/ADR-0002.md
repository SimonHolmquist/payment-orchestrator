# ADR-0002: Domain Events

## Status
Accepted

## Context
Cambios de estado de Payment deben disparar efectos secundarios: publicar eventos, auditar, reconciliar, notificar.

## Decision

PaymentCreated, Authorized, Captured, Refunded, Cancelled, Failed, MarkedUnknown.
